Newsletter Sign-Up Form

Description
-----------------
This project is a simple Newsletter Sign-Up Form with basic email validation, error message display on input error, and a success message displayed upon successful submission. The form is created using HTML, CSS, and JavaScript.

Features
----------
User-friendly Newsletter Sign-Up Form
Basic email validation to ensure the correct email format is entered
Error message display for invalid email input
Success message displayed upon successful submission
Dismiss button to clear error or success messages

How to Use
----------------
Clone the repository using the following command:
bash
Copy code
git clone https://github.com/Vishal-Cr/Newsletter-Sign-Up-Form.git
-------------------
Navigate to the project directory and open the index.html file in your web browser.

Enter your email address in the provided input field and click the "Subscribe" button to submit the form.

If you enter an invalid email format, an error message will be displayed below the input field.

If you successfully submit the form, a success message will be displayed, and you can dismiss it by clicking the "Dismiss" button.
